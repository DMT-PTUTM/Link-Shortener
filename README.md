## Link Shortener for PTUTM
### A telegram bot to shorten long links with PTUTM Domain
## Dependencies
- python telegram wrapper\
  ```pip3 install python-telegram-bot```

## Available commands
- ```/start``` - Starts the bot.

- ```/help``` - For help message.
